# weGPA

## About the Project

weGPA is a dynamic GPA and CGPA calculator designed specifically for college students. It offers a user-friendly platform to accurately calculate GPA and CGPA with the added convenience of exporting results to a PDF file. This project aims to simplify the academic planning process by providing an easy-to-use and reliable tool for tracking academic progress.

## Tech Stack

- **Prisma**: Used for backend development, Prisma offers robust database management and seamless data access.
- **Next.js**: Powers the client-side application, providing a fast and efficient user experience with server-side rendering capabilities.
- **Tailwind CSS**: Utilized for styling, Tailwind CSS allows for responsive and customizable design, ensuring a visually appealing and intuitive user interface.

## Features

- **GPA Calculation**: Allows students to calculate their GPA based on their course grades and credits.
- **CGPA Calculation**: Facilitates the computation of cumulative GPA over multiple semesters.
- **PDF Export**: Users can export their GPA and CGPA calculations as a PDF file, making it easy to save and share their academic records.
- **User-Friendly Interface**: Designed with simplicity and ease of use in mind, enhancing the overall user experience.
- **Responsive Design**: Ensures accessibility and a seamless experience across various devices and screen sizes.

## Getting Started

To get a local copy up and running, follow these simple steps:

1. Clone the repository

   git clone <https://github.com/alphadevking/we_gpa.git>

2. Install NPM packages

   npm install

3. Run the application

   npm run dev
